## 快速入门示例

### 1. 基础 Referer 检查（5 分钟）

**提示词：** "如何使用 Nginx 防止盗链？"

**结果：** 可直接使用的 Nginx 配置：
```nginx
location ~* \.(jpg|jpeg|png|gif)$ {
    valid_referers none blocked yourdomain.com *.yourdomain.com;
    if ($invalid_referer) {
        return 403;
    }
}
```

### 2. 基于令牌的保护（15 分钟）

**提示词：** "为 Flask 生成基于令牌的保护，使用用户特定的令牌。"

**结果：** 完整的 Flask 实现，包含令牌生成、验证和中间件。

### 3. 付费内容保护（30 分钟）

**提示词：** "我需要使用 OAuth 和水印来保护课程视频。"

**结果：** 完整的集成指南，包括：
- OAuth 2.0 认证
- 生成包含用户 ID 的令牌
- FFmpeg 水印工作流
- 速率限制和监控

## 重要说明

### ⚠️ 这些是示例性代码
所有代码示例仅供学习模式使用。在生产环境使用前：
- 添加全面的输入验证
- 实施完整的错误处理
- 进行安全审计
- 设置监控和告警
- 使用适当的密钥管理

### 🔒 了解局限性
防盗链可以防止：
- ✅ 来自其他网站的普通盗链
- ✅ 未授权的直接链接
- ✅ 带宽盗用

它不能防止：
- ❌ 截图和重新分发
- ❌ 令牌分享（如果用户选择分享）
- ❌ 对已下载文件的离线访问
- ❌ 内部威胁（开发者、泄露的密钥）

对于完整的内容保护，考虑结合使用：
- DRM（数字版权管理）
- 取证水印
- 法律协议（服务条款、许可）
- 敏感数据的加密
